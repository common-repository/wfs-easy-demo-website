This plugin helps users to easily create a demo site in wordpress blog, based on wordpress menu to create demo menu

== Description ==

Features: 

<ul>
	<li>Creating a demo site in wordpress blog</li>
	<li>Based on wordpress menu to create demo menu</li> 
	<li>Demo website:
	<ul>
		<li>Connecting wordpress menu to use demo menu</li>
		<li>Unlimited level menu for demo menu</li>
		<li>Supporting demo logo</li>
		<li>Supporting adding Google Analysis script</li>
		<li>Connecting page/post (products) when creating menu link, to avoid re-type demo link for each product</li>
	</ul>
	</li> 
</ul>

Visis us: <a href="http://webfacescript.com/category/product-list/product-list-wordpress-plugins/">webfacescript.com</a>

== Installation ==

1. Upload the entire wfs_easy_demo_website folder to the /wp-content/plugins/ directory or Add New Plugin by upload the wfs_easy_demo_website.zip
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'WFS Easy Demo Website' menu in your WordPress admin panel.
